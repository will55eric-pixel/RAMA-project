# RAMA-project
A modular, expandable space habitat and factory system. Free for anyone to use or build on
RAMA Project — Public Domain Modular Space Architecture
RAMA is an open, modular system for building large‑scale orbital and lunar infrastructure using simple, repeatable components.
The project includes hub structures, ring assemblies, tethered factories, lunar starter kits, and support systems — all designed to be built, adapted, or expanded by anyone.

All RAMA concepts, diagrams, and documents are released into the public domain (CC0).
Anyone may use, modify, publish, or commercialize any part of this work without permission or restriction.

Purpose
RAMA is designed to be:

Modular — built from repeatable sections

Scalable — grows as needed, no fixed size

Repairable — every part accessible and replaceable

Human‑readable — simple geometry, simple logic

Open — no ownership, no patents, no gatekeeping

The goal is to provide a clean, practical foundation for orbital habitats, factories, and lunar operations that anyone can build on.

Why RAMA Exists
RAMA is built on a simple idea:
Space infrastructure should be modular, repairable, and understandable by ordinary people.

No black boxes.
No proprietary systems.
No fragile megastructures.
Just clean geometry, simple mechanics, and scalable logic.
